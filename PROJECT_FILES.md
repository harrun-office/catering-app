# CaterHub Application Files

This folder contains all the necessary files for the CaterHub real-time catering application.

## Quick Navigation

- **backend/**: Express.js server with REST API
- **frontend/**: React.js client application
- **README.md**: Complete project documentation
- **SETUP_GUIDE.md**: Step-by-step setup instructions
- **.gitignore**: Git ignore rules

## Starting the Application

1. **Setup Database**: `mysql < backend/src/utils/database.sql`
2. **Start Backend**: `cd backend && npm install && npm run dev`
3. **Start Frontend**: `cd frontend && npm install && npm start`

For detailed instructions, see **SETUP_GUIDE.md**
